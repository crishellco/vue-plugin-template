## Install

```bash
yarn add -D vue-plugin-template
# or
npm i -D vue-plugin-template
```

```javascript
import Vue from 'vue';
import VuePluginTemplate from 'vue-plugin-template';

Vue.use(VuePluginTemplate);
```
